# APCEM-Enhanced Cervical Cytology Image Dataset
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18860357.svg)](https://doi.org/10.5281/zenodo.18860357)
This repository contains preprocessing, training, and evaluation scripts
corresponding to the APCEM cervical cytology dataset.
This repository provides preprocessing, training, and evaluation scripts associated with the APCEM-Enhanced Cervical Cytology Image Dataset. The repository supports reproducible experiments for cervical cytology image analysis and visual quality assessment.

# Dataset

The APCEM-Enhanced Cervical Cytology Image Dataset is publicly available through Mendeley Data.

Dataset DOI:
https://doi.org/10.17632/snzb5g6ddy.2

The dataset contains enhanced cervical cytology images designed to support research in image preprocessing, visual quality assessment, and machine learning–based analysis of cervical cytology data.

# Repository Purpose

This repository provides the preprocessing pipeline, model training scripts, and evaluation procedures used in experiments conducted with the APCEM-Enhanced Cervical Cytology Image Dataset. The code is designed to facilitate reproducible experimentation.

# Environment

Python 3.10
TensorFlow 2.x

Hardware used for experiments:
NVIDIA Tesla T4 GPU (Google Colab)

# Installation

Install the required dependencies:
pip install -r requirements.txt

# Reproduction

To reproduce the training experiments:
python train.py
