# APCEM-Enhanced Cervical Cytology Image Dataset Code (v1.0)

This repository contains preprocessing, training, and evaluation scripts
corresponding to the APCEM-Enhanced Cervical Cytology Image Dataset.

Dataset DOI:
https://doi.org/10.17632/snzb5g6ddy.2

Environment:
Python 3.10
TensorFlow 2.x
GPU: NVIDIA Tesla T4 (Google Colab)

Reproduction:
pip install -r requirements.txt
python train.py
